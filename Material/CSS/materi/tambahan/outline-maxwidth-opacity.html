Advance Utility: Outline, Max-width, & Opacity

1. Outline (Garis Luar)

Outline adalah garis yang digambar di luar batas (border) elemen. Berbeda dengan border, outline tidak mengambil ruang dalam layout (tidak menambah lebar/tinggi elemen).

Perbedaan Utama dengan Border:

  • Outline digambar di luar border.
  • Outline tidak mempengaruhi ukuran kotak elemen.
  • Outline bisa menggunakan outline-offset untuk memberi jarak dari border.
Kotak ini punya Border Biru dan Outline Merah dengan Offset 5px. 
.box {
  border: 3px solid blue;
  outline: 5px solid red;
  outline-offset: 5px; /* Jarak antara border dan outline */
}
Kasus Penting: Aksesibilitas. Browser memberikan outline otomatis saat Anda menekan tombol 'Tab' pada keyboard. Jangan pernah menghapus outline: none tanpa memberikan gaya fokus pengganti, karena pengguna keyboard akan kesulitan navigasi.

2. Max-width (Batas Lebar Maksimal)

Max-width digunakan untuk mencegah elemen menjadi lebih lebar dari nilai yang ditentukan, tetapi elemen tetap bisa mengecil jika layarnya menyempit.

Kasus Nyata: Desain Responsif

Jika Anda menggunakan width: 800px, website akan terpotong di HP. Jika menggunakan max-width: 800px, website akan mengikuti lebar HP tapi tidak akan lebih dari 800px di laptop.

Kotak ini lebarnya 100%, tapi max-width cuma 500px. Coba kecilkan jendela browser Anda! 
.container {
  width: 100%;      /* Mencoba penuh */
  max-width: 800px;  /* Tapi dilarang lebih dari 800px */
  margin: 0 auto;    /* Mengetengahkan kotak */
}

3. Opacity (Transparansi)

Opacity mengatur tingkat transparansi elemen. Nilainya berkisar dari 0.0 (transparan penuh) hingga 1.0 (solid/tidak transparan).

1.0
0.7
0.4
0.1
Peringatan: Saat Anda memberikan opacity pada elemen induk, maka semua anak elemen di dalamnya (teks, gambar) juga akan ikut transparan.

Kasus Nyata: Efek Hover Gambar

Hover Saya!

.gallery-img {
  opacity: 0.6;
  transition: 0.3s;
}

.gallery-img:hover {
  opacity: 1; /* Gambar menjadi jelas saat disentuh */
}
