Interactive UI: Tooltips, Buttons, & Pagination

1. Tooltips

Tooltip adalah pesan kecil yang muncul saat pengguna mengarahkan kursor (hover) ke sebuah elemen. Fungsinya memberikan konteks tambahan tanpa memenuhi layar.

Demo Visual:

Arahkan kursor pada teks ini:

Dekatkan mouse ke sini Halo! Saya Tooltip 

/* Kunci Tooltip: Position Relative pada Parent 
   dan Position Absolute pada Tooltiptext */
.tooltip .tooltiptext {
    position: absolute;
    bottom: 125%; 
    left: 50%;
    visibility: hidden;
}
.tooltip:hover .tooltiptext {
    visibility: visible;
}

2. Buttons (Tombol)

Tombol adalah elemen interaksi utama. Desain tombol yang baik harus memiliki status **Normal**, **Hover** (saat disentuh), dan **Active** (saat diklik).

Contoh Visual:

.btn {
    transition: 0.3s; /* Agar animasi hover halus */
}
.btn:hover {
    transform: translateY(-3px); /* Efek mengangkat */
    box-shadow: 0 5px 15px rgba(0,0,0,0.2);
}
Tips: Gunakan cursor: pointer; agar pengguna tahu bahwa elemen tersebut bisa diklik.

3. Pagination (Penomoran Halaman)

Pagination digunakan jika Anda memiliki daftar konten yang sangat banyak (seperti hasil pencarian Google) agar beban loading tidak berat.
