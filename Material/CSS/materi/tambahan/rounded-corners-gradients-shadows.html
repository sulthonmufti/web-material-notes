Visual Aesthetics: Rounded, Gradients, & Shadows

1. Rounded Corners (border-radius)

Properti border-radius digunakan untuk membuat sudut elemen menjadi tumpul atau bulat.

Istilah Baru:
  • Pixels (px): Memberikan lengkungan yang tetap.
  • Percentage (%): Jika diset 50% pada kotak persegi, elemen akan menjadi lingkaran sempurna.
Sudut Tumpul (20px) 
.button {
    border-radius: 10px; /* Sudut halus */
}

.avatar {
    width: 100px; height: 100px;
    border-radius: 50%; /* Bulat sempurna */
}

2. Gradients (Gradasi Warna)

Gradients adalah transisi halus antara dua warna atau lebih. CSS mengenal dua jenis utama:

  • Linear Gradient: Warna mengalir ke satu arah (atas, bawah, kiri, kanan, atau diagonal).
  • Radial Gradient: Warna menyebar dari titik tengah (melingkar).

Kasus Nyata: Background Hero Section

Linear Gradient 135deg 
/* Arah, Warna 1, Warna 2 */
.hero {
    background: linear-gradient(to right, #6c5ce7, #00cec9);
}

.circle {
    background: radial-gradient(circle, #fab1a0, #e17055);
}

3. Shadows (Efek Bayangan)

Shadow memberikan efek "kedalaman" sehingga elemen terlihat menonjol dari latar belakang. Ada dua jenis:

  • Box Shadow: Bayangan untuk kotak/elemen.
  • Text Shadow: Bayangan khusus untuk teks.

Anatomi Box Shadow:

box-shadow: 10px 10px 20px rgba(0,0,0,0.2);

  1. 10px (X): Geser bayangan ke kanan.
  2. 10px (Y): Geser bayangan ke bawah.
  3. 20px (Blur): Seberapa kabur bayangannya.
  4. rgba: Warna bayangan (disarankan menggunakan transparansi/alpha).
Floating Effect (Box Shadow) 
/* Memberikan kedalaman pada kartu */
.card {
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

/* Memberikan efek cahaya pada teks */
h1 {
    text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
}
