Advance CSS Mastery

Image Styling & Filters

Mengubah gambar mentah menjadi elemen estetis menggunakan manipulasi CSS.

Kasus Terpopuler: Instagram-like Filters

Kita bisa memberikan efek filter dan bentuk hanya dengan satu baris kode.

Filter: Sepia & Contrast

img {
    border-radius: 50%;    /* Lingkaran */
    object-fit: cover;     /* Anti Gepeng */
    filter: grayscale(100%); /* Hitam Putih */
}

Modern Modals & Overlays

Modal digunakan untuk interaksi krusial tanpa meninggalkan halaman utama.

Kasus Nyata: Confirm Delete Box

Hapus file ini?

.modal-overlay {
    position: fixed; top: 0; left: 0;
    width: 100vw; height: 100vh;
    background: rgba(0,0,0,0.7); /* Backdrop gelap */
    z-index: 999;
}

Glassmorphism & Gradients

Tren desain modern yang memberikan efek "kaca buram" (frosted glass) di atas latar belakang warna-warni.

Kasus Nyata: Card iOS Style

Glass Card

Sangat populer untuk dashboard modern.

.glass-card {
    background: rgba(255, 255, 255, 0.2);
    backdrop-filter: blur(10px); /* Efek Buram Kaca */
    border: 1px solid rgba(255, 255, 255, 0.3);
}

CSS Animations & Transitions

Membuat website terasa hidup dan interaktif.

Kasus Nyata: Tombol Berdenyut (Pulse)

@keyframes pulse {
    from { transform: scale(1); }
    to { transform: scale(1.1); }
}
button:hover { animation: pulse 1s infinite alternate; }
