Media Queries

1. Konsep Dasar & Viewport

Media Queries memungkinkan kita menerapkan gaya CSS yang berbeda berdasarkan karakteristik perangkat, seperti lebar layar, orientasi (portrait/landscape), atau resolusi.

Viewport: Area layar yang terlihat oleh pengguna. Di HTML, kita wajib memasang tag <meta name="viewport" ...> di bagian <head> agar Media Queries bekerja dengan benar di HP.

2. Penulisan Syntax

Media query ditulis dengan simbol @media diikuti oleh kondisi tertentu.

@media screen and (max-width: 600px) {
  /* CSS di sini hanya jalan jika lebar layar 
     600px atau lebih kecil */
  body {
    background-color: orangered;
  }
}

3. Apa itu Breakpoints?

Breakpoints adalah titik koordinat lebar layar (dalam pixel) di mana susunan elemen website Anda berubah total. Di titik inilah kita memberitahu browser: "Jika layar sesempit ini, ubah tampilannya!"

Penting: Jangan membuat terlalu banyak breakpoint. Fokuslah pada konten. Jika tampilan mulai terlihat berantakan pada lebar tertentu, di situlah Anda butuh breakpoint.

Standar Ukuran Industri:

Kategori Lebar (px) Breakpoint Umum
Mobile (HP) 320px - 480px 480px
Tablet 481px - 768px 768px
Laptop 769px - 1024px 1024px

Contoh Kasus: Perubahan Layout Kolom

Misalnya kita punya 3 kotak. Di Desktop kita ingin berjajar menyamping (3 kolom), tapi di HP kita ingin menumpuk ke bawah (1 kolom).

Kode Breakpoint-nya:

/* Tampilan Default (Desktop) */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

/* Breakpoint untuk Layar Kecil (Mobile) */
@media (max-width: 600px) {
  .container {
    grid-template-columns: 1fr; /* Berubah jadi satu kolom penuh */
  }
}
Tips Visual: Saat Anda melakukan inspeksi (F12) di browser, gunakan tombol "Toggle Device Toolbar" (icon HP/Tablet) untuk menggeser lebar layar secara manual dan melihat kapan desain Anda berubah.

4. Strategi Mobile First

Saat ini, developer lebih suka menggunakan strategi Mobile First. Artinya, kita menulis CSS untuk layar kecil dulu, baru kemudian menggunakan media query min-width untuk layar yang lebih besar.

/* CSS Standar (Untuk HP) */
.container { width: 100%; }

/* Untuk Tablet ke atas */
@media (min-width: 768px) {
  .container { width: 750px; }
}

/* Untuk Desktop ke atas */
@media (min-width: 1024px) {
  .container { width: 960px; }
}
Keuntungan: Website terasa lebih ringan di HP karena browser tidak perlu membaca banyak aturan CSS yang berat dari awal.
