CSS Variables (Custom Properties)

Apa itu CSS Variables?

CSS Variables adalah entitas yang didefinisikan oleh penulis CSS yang berisi nilai spesifik untuk digunakan kembali di seluruh dokumen.

Bayangkan Anda memiliki warna biru yang sama di 50 tempat berbeda. Tanpa variabel, jika Anda ingin mengubah biru tersebut menjadi hijau, Anda harus mencarinya satu per satu. Dengan variabel, Anda cukup mengubah satu baris kode.

1. Cara Mendeklarasikan Variabel

Variabel biasanya dideklarasikan di dalam pseudo-class :root agar bisa diakses oleh seluruh elemen di halaman (Global).

Aturan Penulisan: Nama variabel harus dimulai dengan dua tanda hubung (--). 
:root {
  --main-bg-color: coral;
  --main-txt-color: white;
  --padding-standard: 15px;
}

2. Cara Menggunakan Variabel (Fungsi var())

Untuk mengambil nilai dari variabel, kita menggunakan fungsi var().

body {
  background-color: var(--main-bg-color);
  color: var(--main-txt-color);
  padding: var(--padding-standard);
}

Kasus Nyata: Dark Mode (Mode Gelap)

Ini adalah kegunaan variabel yang paling populer. Kita hanya perlu mengubah nilai variabel di dalam sebuah class "dark-theme".

:root {
  --bg-color: white;
  --text-color: black;
}

/* Saat class .dark-mode aktif pada body */
body.dark-mode {
  --bg-color: #1a1a1a;
  --text-color: #f0f0f0;
}

/* Elemen tetap menggunakan var() yang sama */
section {
  background-color: var(--bg-color);
  color: var(--text-color);
}

Kelebihan Menggunakan Variabel

  • Maintenance Mudah: Ubah di satu tempat, berubah di semua tempat.
  • Keterbacaan: Nama --brand-red lebih mudah dipahami dibanding #e74c3c.
  • Dinamis: Bisa diubah menggunakan JavaScript secara real-time.
