Memahami CSS Box Sizing

Apa itu Box Sizing?

Box Sizing adalah properti CSS yang menentukan bagaimana ukuran total sebuah elemen (lebar dan tinggi) dihitung oleh browser.

Secara default, jika Anda memberi lebar 100px pada kotak, lalu menambah padding 20px, kotak tersebut akan membengkak menjadi 140px. Box sizing hadir untuk mengontrol "pembengkakan" tersebut.

1. content-box (Nilai Bawaan)

Ini adalah pengaturan standar browser. Ukuran width dan height hanya berlaku untuk isinya saja.

Rumus Total Lebar:
Width + Padding + Border = Total Lebar Elemen
Content-box
Width: 300px + Padding 20px + Border 10px = 360px! 
.box {
  width: 300px;
  padding: 20px;
  border: 10px solid black;
  box-sizing: content-box; /* Ukuran asli membengkak */
}

2. border-box (Pilihan Profesional)

Dengan border-box, ukuran width dan height sudah mencakup padding dan border. Kotak tidak akan membengkak.

Rumus Total Lebar:
Width = Total Lebar Elemen (Padding & Border memakan ruang dari dalam)
Border-box
Width: Tetap 300px! (Padding & Border masuk ke dalam) 
.box {
  width: 300px;
  padding: 20px;
  border: 10px solid black;
  box-sizing: border-box; /* Ukuran tetap konsisten */
}

Kasus Nyata: Kenapa Border-box Penting?

Bayangkan Anda membuat dua kotak berdampingan dengan lebar masing-masing 50%.

  • Jika pakai Content-box: Saat Anda menambah padding, lebarnya menjadi 50% + padding. Hasilnya? Kotak kedua akan jatuh ke bawah karena total lebar melebihi 100%.
  • Jika pakai Border-box: Meskipun Anda menambah padding, lebarnya tetap 50%. Kotak akan tetap rapi berdampingan.
Praktik Terbaik: Hampir semua web developer modern mereset semua elemen agar menggunakan border-box di awal kode CSS mereka. 
/* "Global Reset" yang sering dipakai profesional */
* {
  box-sizing: border-box;
}

/* Kode CSS untuk Content-box */
.layout-box {
    width: calc(50% - 5px); /* target width */
    padding: 15px;
    border: 5px solid gray;
    box-sizing: content-box; /* total width = 50% + 30px padding + 10px border */
}

/* Kode CSS untuk Border-box */
.layout-box.is-border-box {
    width: calc(50% - 5px); /* target width */
    padding: 15px;
    border: 5px solid gray;
    box-sizing: border-box; /* total width TETAP 50% */
}
