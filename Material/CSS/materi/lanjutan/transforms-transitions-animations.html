CSS Transitions, Transforms, & Animations

1. CSS Transitions

Transition adalah "jembatan" yang menghaluskan perubahan properti CSS dari kondisi A ke kondisi B (misal: saat tombol di-hover).

Properti Utama:

  • transition-property: Apa yang mau diubah? (Contoh: background-color, all).
  • transition-duration: Berapa lama perubahannya? (Contoh: 0.5s).
  • transition-timing-function: Irama gerakannya (Contoh: ease, linear).
Analogi: Transisi adalah seperti perpindahan gigi mobil yang halus, bukan langsung loncat kecepatan.
Hover Me!
.box {
  transition: all 0.5s ease-in-out;
}
.box:hover {
  background-color: #e67e22;
  border-radius: 50%;
}

2. 2D & 3D Transforms

Transform digunakan untuk mengubah bentuk, posisi, dan ukuran elemen tanpa mengganggu posisi elemen di sekitarnya.

Fungsi Populer (2D):

  • translate(x, y): Memindahkan posisi.
  • rotate(deg): Memutar elemen (Contoh: 45deg).
  • scale(n): Mengubah ukuran (Contoh: 1.5 berarti 150%).
  • skew(deg): Menarik sudut elemen (miring).

Arahkan kursor (hover) pada kotak di bawah untuk melihat fungsinya secara langsung:

Translate(20, -10)
Rotate(45deg)
Scale(1.3)
Skew(20deg)

Kode CSS Dasar:

/* 1. Berpindah: Kanan 20px, Atas 10px */
.item:hover { transform: translate(20px, -10px); }

/* 2. Berputar searah jarum jam */
.item:hover { transform: rotate(45deg); }

/* 3. Membesar 1.3 kali lipat */
.item:hover { transform: scale(1.3); }

/* 4. Miring (Distorsi) */
.item:hover { transform: skew(20deg); }

Dunia 3D:

Dengan menambahkan perspective, kita bisa memutar elemen di sumbu Z (seolah mendekat/menjauh dari layar).

Rotate
.box:hover {
  transform: rotate(45deg) scale(1.1);
}

3. CSS Animations (@keyframes)

Animation jauh lebih kuat dari transisi karena bisa bergerak sendiri tanpa perlu interaksi user (hover) dan memiliki banyak langkah.

Langkah 1: Membuat Storyboard (@keyframes)

@keyframes goyang {
  0% { transform: translateX(0); }
  50% { background: green; }
  100% { transform: translateX(100px); }
}

Langkah 2: Memanggil Animasi

.anim-box {
  animation: goyang 2s infinite;
}
Goyang
Istilah Baru: iteration-count. Gunakan nilai infinite agar animasi berulang selamanya.
