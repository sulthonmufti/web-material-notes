CSS Grid Layout

1. Konsep Dasar & Istilah Baru

Sebelum mulai menulis kode, Anda harus memahami anatomi dari sebuah Grid:

  • Grid Container: Elemen induk tempat display: grid; diterapkan.
  • Grid Item: Elemen anak langsung dari container.
  • Grid Line: Garis pembagi antar kolom atau baris (seperti garis pada buku kotak-kotak).
  • Grid Track: Ruang di antara dua garis grid (bisa berupa satu kolom atau satu baris penuh).
  • Grid Cell: "Satu kotak" terkecil di dalam grid.

2. Membuat Kolom dan Baris

Kita menentukan struktur grid menggunakan grid-template-columns dan grid-template-rows.

.container {
  display: grid;
  grid-template-columns: 200px 200px 200px; /* 3 kolom masing-masing 200px */
  grid-template-rows: 100px 100px;           /* 2 baris masing-masing 100px */
}

3. Satuan FR (Fractional Unit)

Istilah baru: FR. Ini adalah satuan fleksibel yang mewakili "satu bagian dari ruang yang tersisa".

Sangat mirip dengan flex-grow. Jika Anda punya 1fr 2fr 1fr, kolom tengah akan mengambil ruang 2x lebih banyak daripada kolom lainnya. 
grid-template-columns: 1fr 2fr 1fr;

Fungsi Repeat()

Daripada menulis 1fr 1fr 1fr, gunakan: repeat(3, 1fr);

4. Gap (Gutter)

Sebelumnya, kita memberikan jarak antar kotak menggunakan margin, namun ini sering menyebabkan masalah karena sisi luar kotak juga ikut berjarak. Gap hanya memberikan jarak **di antara** item (selokan), tanpa mempengaruhi sisi luar container.

Istilah Spesifik Gap:

  • row-gap: Mengatur jarak horizontal antar baris.
  • column-gap: Mengatur jarak vertikal antar kolom.
  • gap: Shorthand untuk mengatur keduanya sekaligus (gap: [row] [column]).
Analogi: Bayangkan ubin lantai. gap adalah semen pengisi (nat) di antara ubin. Semen itu tidak ada di pinggir tembok, hanya ada di sela-sela antar ubin.

5. Grid Placement (Garis Grid)

Di CSS Grid, posisi item ditentukan oleh Grid Lines (Garis Grid). Garis ini adalah pembatas antar kotak yang memiliki nomor urut.

Penting: Jika Anda punya 3 kolom, maka ada 4 garis.
Garis 1 (paling kiri), Garis 2, Garis 3, dan Garis 4 (paling kanan).

Properti Utama:

  • grid-column-start / grid-column-end: Menentukan dari garis mana ke garis mana item melebar secara horizontal.
  • grid-row-start / grid-row-end: Menentukan posisi secara vertikal.
  • Shorthand: grid-column: 1 / 4; (Artinya dari garis 1 sampai 4).

Contoh Visual:

.header { grid-column: 1 / 4; }
.sidebar { grid-row: 2 / 4; }
.konten-c { grid-column: 2 / 4; }

6. Grid Area (Layout dengan Nama)

Ini adalah cara paling modern dan intuitif. Alih-alih menghitung angka garis, kita memberikan nama pada setiap area dan "menggambar" layoutnya.

Langkah 1: Beri Nama pada Item

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: konten; }
.footer { grid-area: footer; }

Langkah 2: Gambar Layout di Container

Kita menggunakan properti grid-template-areas untuk menyusun kata-kata tersebut menjadi bentuk visual.

.container {
  display: grid;
  grid-template-columns: 1fr 3fr;
  grid-template-areas: 
    "header header"
    "sidebar konten"
    "footer footer";
}
