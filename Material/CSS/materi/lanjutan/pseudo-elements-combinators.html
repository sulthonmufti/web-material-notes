Pseudo-classes, Elements & Combinators

1. Pseudo-classes (Simbol :)

Pseudo-class digunakan untuk mendefinisikan keadaan khusus (state) dari suatu elemen.

Yang Sering Digunakan:

  • :hover : Saat kursor mouse di atas elemen.
  • :active : Saat elemen sedang diklik.
  • :focus : Saat elemen (seperti input) sedang dipilih/diketik.
  • :nth-child(n) : Memilih anak ke-sekian (Contoh: memilih baris ganjil di tabel).

Demo Visual:

A. State Interaksi (Hover, Active, Focus)

Cobalah berinteraksi dengan elemen-elemen di bawah ini untuk melihat kodenya bekerja:

/* Saat kursor di atas tombol */
    .btn:hover { background: #2ecc71; }

    /* Saat tombol ditekan/klik tahan */
    .btn:active { transform: translateY(4px); }

    /* Saat input sedang aktif (diketik) */
    .input:focus { border-color: #27ae60; }

B. Structural Pseudo-classes (nth-child)

Ini digunakan untuk memilih elemen berdasarkan posisinya dalam daftar. Sangat berguna untuk membuat tabel "Zebra" agar mudah dibaca.

  • Baris 1 (Ganjil - Background Abu)
  • Baris 2 (Anak kedua - Warna Merah)
  • Baris 3 (Ganjil - Background Abu)
  • Baris 4 (Genap - Putih)
  • Baris 5 (Anak Terakhir - Warna Hijau)
/* Memilih baris ganjil (1, 3, 5, dst) */
li:nth-child(odd) { background: #eee; }

/* Memilih baris genap (2, 4, 6, dst) */
li:nth-child(even) { background: #fff; }

/* Memilih anak ke-2 secara spesifik */
li:nth-child(2) { color: red; }

/* Memilih anak pertama dan terakhir */
li:first-child { font-weight: bold; }
li:last-child { color: green; }

2. Pseudo-elements (Simbol ::)

Pseudo-element digunakan untuk membuat elemen "palsu" yang tidak ada di HTML, tetapi muncul di layar untuk keperluan dekorasi atau informasi tambahan.

Kasus yang Sering Digunakan di Industri:

A. Membuat Garis Dekorasi (Modern Heading)

Daripada membuat <div> kosong hanya untuk sebuah garis, gunakan ::after. Ini menjaga HTML Anda tetap bersih.

Layanan Kami

.fancy-header::after {
    content: "";
    position: absolute;
    bottom: -5px;
    width: 50%;
    height: 4px;
    background: #27ae60;
}

B. Custom List Bullets (Ikon Daftar)

Mengganti titik (bullet) standar yang membosankan dengan ikon atau simbol unik menggunakan ::before.

  • Gratis Konsultasi
  • Update Selamanya
  • Dukungan 24/7
.custom-list li::before {
    content: "✔";
    color: #27ae60;
    position: absolute;
    left: 0;
}

C. Efek Hover Berjalan (Advanced)

Pseudo-element sering digunakan sebagai "lapisan latar" yang bergerak saat tombol disentuh.

/* Lapisan latar belakang yang bergeser */
.btn-slide::before {
    content: "";
    position: absolute;
    left: -100%; /* Sembunyi di kiri */
    width: 100%; height: 100%;
    background: #27ae60;
    transition: 0.4s;
}
.btn-slide:hover::before { left: 0; }
Pelajaran Penting: ::before dan ::after sangat bergantung pada position: absolute;. Oleh karena itu, elemen induknya (seperti .btn-slide atau li) wajib memiliki position: relative; agar elemen palsu tersebut tidak lari ke luar kotak.

Tabel Referensi Pseudo-element

Berikut adalah daftar Pseudo-element lainnya beserta tingkat prioritas penggunaannya dalam pengembangan web:

Pseudo-element Keterangan Prioritas
::after Menyisipkan konten setelah isi elemen ★★★★★
::backdrop Backdrop pada elemen <dialog> ★★★
::before Menyisipkan konten sebelum isi elemen ★★★★★
::cue Untuk styling cue (teks subtitle) pada video (WebVTT) ★★
::details-content Styling konten pada elemen <details> ★★★
::file-selector-button Styling tombol pada <input type="file"> ★★★★
::first-letter Menargetkan huruf pertama dalam paragraf ★★★
::first-line Menargetkan baris pertama dalam teks ★★★
::marker Menentukan gaya bullet atau numbering pada list ★★★★
::selection Menentukan gaya teks yang diseleksi user ★★★★★
::target-text Menyorot teks target fragment URL ★★★
::view-transition Memberikan animasi transisi halaman ★★★★

3. Combinators (Hubungan Antar Elemen)

Combinator menjelaskan hubungan antara dua selector. Ini sangat berguna untuk menjaga kode CSS tetap bersih dan efisien.

Simbol Nama Analogi Sederhana
(spasi) Descendant Semua keturunan (Anak, Cucu, Cicit).
> Child Hanya Anak Kandung (Level 1).
+ Adjacent Sibling Adik tepat di bawah kakak.
~ General Sibling Semua adik-adik di bawah kakak.

Kasus Nyata & Demo Visual:

A. Descendant (Spasi) vs Child (>)

Gunakan > jika Anda ingin gaya yang sangat spesifik untuk level pertama saja.

Saya Anak Langsung (Orange & Bold)

Saya Cucu (Abu-abu saja)

/* Semua p di dalam box (anak & cucu) */
.family-box p { color: gray; }

/* Hanya p di level pertama */
.family-box > p { color: orange; font-weight: bold; }

B. Menghapus Jarak Antar Judul (+)

Kasus tersering: Memberikan jarak otomatis pada paragraf yang berada tepat setelah judul.

Judul Artikel

Ini paragraf pertama (Tepat setelah judul, jadi miring & biru).

Ini paragraf kedua (Biasa saja).

/* Targetkan p yang ada tepat di bawah h4 */
h4 + p { 
    font-style: italic; 
    color: blue; 
}

C. General Sibling Selector (~)

Digunakan untuk menandai semua elemen yang setingkat di bawah elemen tertentu. Berguna untuk fitur "Checklist" atau navigasi.

Belajar Combinators
Mengerjakan Tugas 
/* Jika input di-check, coret teks span di sebelahnya */
input:checked ~ span {
    text-decoration: line-through;
}
Tips Penting: Sibling selector (+ dan ~) hanya bekerja untuk elemen yang berada setelah/di bawah elemen referensi dalam kode HTML. CSS tidak bisa memilih "kakak" (elemen di atasnya).
