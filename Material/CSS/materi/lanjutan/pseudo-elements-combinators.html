Pseudo-classes, Elements & Combinators

1. Pseudo-classes (Simbol :)

Pseudo-class digunakan untuk mendefinisikan keadaan khusus (state) dari suatu elemen.

Yang Sering Digunakan:

  • :hover : Saat kursor mouse di atas elemen.
  • :active : Saat elemen sedang diklik.
  • :focus : Saat elemen (seperti input) sedang dipilih/diketik.
  • :nth-child(n) : Memilih anak ke-sekian (Contoh: memilih baris ganjil di tabel).

Demo Visual:

button:hover { background: darkblue; }
button:active { transform: scale(0.9); }

2. Pseudo-elements (Simbol ::)

Pseudo-element digunakan untuk memberi gaya pada bagian tertentu dari suatu elemen.

Yang Sering Digunakan:

  • ::before : Menambahkan konten sebelum isi elemen.
  • ::after : Menambahkan konten sesudah isi elemen.
  • ::selection : Mengubah gaya teks saat di-highlight (blok) oleh user.

3. Combinators (Hubungan Antar Elemen)

Combinator menjelaskan hubungan antara dua selector. Ini sangat berguna untuk menjaga kode CSS tetap bersih.

Simbol Nama Penjelasan
(spasi) Descendant Pilih semua anak, cucu, cicit di dalam.
> Child Selector Hanya anak langsung (level 1).
+ Adjacent Sibling Elemen yang tepat berada di bawahnya.

Contoh Visual:

Anak Langsung (Warna Oranye - pakai > )

Cucu (Warna Abu-abu - pakai spasi)
