CSS Flexbox (Flexible Box)

Flexbox Intro

Flexbox adalah model layout satu dimensi (baris atau kolom). Sebelum Flexbox, kita menggunakan float yang sangat rumit.

Cara Mengaktifkan:

Anda cukup memberikan properti display: flex; pada elemen induknya (container).

.container {
  display: flex;
  flex-direction: row; /* Default: item berjejer ke samping */
}

Justify Content

Properti ini mengatur perataan item di sepanjang **Main Axis** (secara default: Horisontal).

  • flex-start (Default): Rata kiri.
  • flex-end: Rata kanan.
  • center: Di tengah.
  • space-between: Jarak merata di antara item.
  • space-around: Jarak merata di sekeliling item.

Flex Items (Properties untuk Anak)

Selain container, kita bisa mengatur masing-masing item di dalamnya.

  • flex-grow: Menentukan seberapa besar item akan tumbuh dibanding item lain.
  • flex-shrink: Menentukan kemampuan item untuk mengecil.
  • flex-basis: Menentukan ukuran awal item.

Align Items & Perilaku Item (Vertikal)

Bagian ini membahas cara menyelaraskan elemen secara vertikal dan bagaimana elemen tersebut bereaksi terhadap ruang yang tersedia.

A. Penyelarasan Vertikal (Align Items)

  • stretch (Default): Item memenuhi tinggi container.
  • center: Item berada di tengah vertikal.
  • flex-start: Item di atas.
  • flex-end: Item di bawah.

B. Logika Pertumbuhan Item (Deep Dive)

Saat menyelaraskan secara vertikal, kita sering perlu mengatur bagaimana item mengisi ruang horisontal yang tersisa.

1. flex-grow Tumbuh

Mengatur seberapa besar item boleh "tumbuh" saat ada ruang kosong.

  • Default: 0 (Tidak membesar).
  • Contoh: flex-grow: 2; artinya item tersebut mengambil sisa ruang 2x lebih banyak dibanding yang bernilai 1.

2. flex-shrink Menyusut

Mengatur seberapa besar item boleh "mengecil" saat ruang tidak cukup.

  • Default: 1 (Boleh mengecil).
  • Nilai 0: Item mempertahankan ukurannya (sangat berguna untuk Icon/Sidebar).

3. flex-basis Modal Awal

Ukuran awal item sebelum grow atau shrink dihitung. Anggap ini sebagai titik start elemen.

Responsive Flexbox

Salah satu kekuatan Flexbox adalah kemampuannya berubah dari baris (row) menjadi kolom (column) pada layar kecil.

.container {
  display: flex;
  flex-direction: row;
}

/* Jika layar di bawah 600px, berubah jadi kolom */
@media (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}
Coba kecilkan ukuran browser Anda (jika menggunakan Desktop) untuk melihat efek flex-wrap: wrap; di bawah ini.
