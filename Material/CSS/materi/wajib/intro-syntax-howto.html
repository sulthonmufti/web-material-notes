CSS Introduction, Syntax, dan How To

Selamat datang! Ini adalah langkah pertama Anda dalam menguasai CSS. CSS (Cascading Style Sheets) berfungsi untuk mengatur tampilan dan presentasi elemen-elemen HTML Anda.

1. CSS Introduction (Peran CSS)

CSS adalah bahasa yang digunakan untuk mendeskripsikan gaya dari dokumen HTML. Tanpa CSS, halaman web hanya akan terlihat seperti dokumen teks biasa.

  • **Tugas utama CSS:** Mengatur warna, font, layout (tata letak), dan animasi.
  • **Prinsip Dasar:** Memisahkan konten (HTML) dari presentasi (CSS) untuk mempermudah pemeliharaan dan perubahan desain.

2. CSS Syntax (Aturan Penulisan)

Setiap aturan CSS terdiri dari dua bagian utama: **Selector** dan **Declaration Block**.

Struktur Dasar


selector {
    property: value;
    property: value; 
}

Penjelasan:

  • **Selector:** Menunjuk elemen HTML mana yang ingin Anda beri gaya (misalnya, p untuk paragraf, h1 untuk judul).
  • **Declaration Block:** Berada di dalam kurung kurawal ({}), berisi satu atau lebih deklarasi yang dipisahkan oleh titik koma (;).
  • **Declaration:** Adalah pasangan **Property** (properti) dan **Value** (nilai) yang dipisahkan oleh titik dua (:).

Contoh Nyata

Jika kita ingin mengubah warna semua paragraf menjadi biru dan ukuran font menjadi 16px:


p {
    color: blue;
    font-size: 16px;
}

3. CSS How To (Cara Menghubungkan CSS)

Ada tiga cara utama untuk menerapkan CSS ke dokumen HTML Anda. **Paling direkomendasikan adalah External CSS.**

1. External CSS (Paling Direkomendasikan)

Membuat file CSS terpisah (misalnya, style.css) dan menautkannya di bagian <head> HTML.


<head>
<link rel="stylesheet" href="style.css">
</head>

Ini adalah cara yang paling bersih dan efisien untuk proyek besar.

2. Internal CSS

Menulis kode CSS langsung di dalam tag <style> di bagian <head> dokumen HTML.


<head>
<style>
body { background-color: lightgray; }
</style>
</head>

3. Inline CSS (Hindari untuk Styling Massal)

Menulis gaya CSS langsung sebagai atribut pada elemen HTML menggunakan atribut style.


 <p style="color: red; margin-left: 20px;">Paragraf merah ini menggunakan Inline CSS.</p>

Setelah menguasai cara menautkan CSS dan memahami sintaks dasarnya, Anda siap untuk mempelajari konsep fundamental **Box Model**!
