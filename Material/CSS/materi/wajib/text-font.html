CSS Text dan Fonts

CSS Text

Properti CSS Text digunakan untuk mengatur tampilan isual dan tata letak teks.

1. Mengatur Alignment (text-align)

Mengatur perataan horizontal text di dalam elemen:

  • left(default)
  • right
  • center
  • justify(Meratakan tepi kiri dan kanan)
Text ini berada di tengah (center)

2. Decoration text-decoration

Menambahkan atau menghapus dekorasi pada teks, sangat umum digunakan untuk emnghilangkan garis bawah pada tautan.

  • none (penting untuk tautan)
  • underline
  • overline
  • line-through
a{text-decoration:none;}
Menghilangkan garis pada link: google.com

3. Transformation text-transform

Mengubah *casing* huruf, terlepas dari bagaimana test ditulis di HTML.

  • uppercase SEMUA HURUF BESAR
  • lowercase semua huruf kecil
  • capitalize Huruf Pertama Setiap Kata Besar
Teks ini akan menjadi huruf besar semua

4. Spacing line-height, letter-spacing, word-spacing

Mengontrol jarak antar baris, antar huruf, dan antar kata.

  • line-height: Jarak antara baris teks (penting untuk keterbacaan)
  • letter-spacing:Jarak anatar karakter
  • word-spacing: Jarak antar kata.
p{ ling-height: 1.5; letter-spacing: 2px; }
Teks ini menggunakan jarak baris teks 1.5 dan jarak antar karakter 2px

5. Shadow text-shadow

Menambahkan bayangan pada teks. Nilai urutannya: **horizontal offset**, **vertical offset, **blur radius**, **color**

text-shadow: 2px 2px 4px #000000
Teks ini dengan bayangan

CSS Fonts

Properti CSS Fonts digunakan untuk mengatur tampilan tipografi teks.

1. Font Famili (font-family)

Menentukan jenis huruf. Sangat penting untuk menentukan **font Fallbacks.

font-family: "Helvetica Neue, Arial, san-serif;

2. Web Safe dan Fallbacks

Browser mungkin memiliki font yang kita inginkan. Karena itu, kita harus selalu menyertakan:

  • **Font Utama** (misalnya, "Helvetica Neue")
  • **Font Alternatif** (misalnya, Arial)
  • ** Generic Family** (serif, sans-serif, monospace) sebagai *fallback* terakhir.
Contoh font: "Courier New", monospace.

3. Style, Size, Weight (font-style, font-size, font-weight)

  • font-style: Noemal, italic, atau oblique.
  • font-size: Ukuran teks (gunakan px, em, atau rem).
  • font-weight: Ketebalan teks (normal, bold, atau nilai angka 100-900.
