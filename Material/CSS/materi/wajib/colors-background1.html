Colors dan Backgrounds CSS

Colors (Warna Teks)

Properti utama untuk mengatur warna teks adalah color.

Metode Penentuan Warna

Ada empat cara utama untuk mendefinisikan warna di CSS:

1. Nama Warna (Color Keywords)

Menggunakan nama standar yang mudah dibaca, misalnya red, blue, tomato.

Teks ini berwarna RED
color: red;

2. Kode HEX (Hexadecimal)

Menggunakan format #RRGGBB. Setiap pasangan mewakili nilai Red, Green, dan Blue dari 00 hingga FF.

Teks ini berwarna HIJAU HEX (#4CAF50)
color: #4CAF50;

3. Nilai RGB (Red, Green, Blue)

Menggunakan format fungsi rgb(R, G, B), dengan nilai dari 0 hingga 255.

Teks ini berwarna TOMATO RGB
color: rgb(255, 99, 71);

4. Nilai HSL (Hue, Saturation, Lightness)

Menggunakan fungsi hsl(H, S, L). Lebih intuitif untuk manusia, memisahkan rona (Hue), kejenuhan (Saturation), dan kecerahan (Lightness).

Teks ini berwarna BIRU HSL
color: hsl(240, 100%, 50%);

CSS Backgrounds (Latar Belakang)

Properti background-* digunakan untuk mengontrol latar belakang elemen.

1. Background Color

Mengatur warna latar belakang elemen. Anda dapat menggunakan semua format warna di atas (nama, HEX, RGB, HSL).

Latar belakang elemen ini berwarna biru pucat (Alice Blue). 
background-color: #f0f8ff;

2. Background Image

Menambahkan gambar sebagai latar belakang.

background-image: url('jalur/ke/gambar.jpg');

3. Background Repeat

Mengontrol bagaimana gambar latar belakang diulang.

  • repeat (default): Mengulang horizontal dan vertikal.
  • no-repeat: Gambar ditampilkan sekali saja.
  • repeat-x: Mengulang hanya secara horizontal.
  • repeat-y: Mengulang hanya secara vertikal.

Contoh no-repeat:

background-repeat: no-repeat;

4. Background Attachment

Menentukan apakah gambar latar belakang akan bergulir (*scroll*) bersama halaman atau tetap (*fixed*).

  • scroll (default): Gambar bergerak saat halaman digulir.
  • fixed: Gambar tetap pada posisinya saat halaman digulir.
background-attachment: fixed;

5. Background Shorthand (Ringkasan)

Cara efisien untuk mengatur semua properti latar belakang dalam satu deklarasi. Urutan tidak terlalu ketat, tetapi disarankan:


/* color image repeat attachment position */
background: #ffffff url("gambar.png") no-repeat fixed center;

Properti ini sangat berguna untuk menulis CSS yang lebih ringkas.
