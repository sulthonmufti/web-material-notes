CSS Units (Satuan)

Absolute Units (Satuan Tetap)

Satuan absolut memiliki ukuran yang sama persis, tidak peduli seberapa besar layar atau pengaturan browser pengguna.

Pixels (px)

  • Satuan paling umum digunakan.
  • 1px setara dengan 1 titik di layar.
  • **Kekurangan:** Tidak fleksibel untuk aksesibilitas. Jika pengguna mengubah ukuran font default di browser mereka, ukuran px tetap tidak akan berubah.

Teks ini berukuran 16px (Tetap).

Relative Units (Satuan Relatif)

Satuan relatif berubah ukurannya berdasarkan elemen lain (induk atau root). Ini sangat disarankan untuk desain modern.

1. EM (Relative to Parent)

Ukuran em relatif terhadap font-size elemen induknya.

Induk (Font size: 20px)
Anak (2em = 40px)

Hati-hati: Jika em digunakan bertingkat (anak di dalam anak), ukurannya bisa menjadi sangat besar/kecil dengan cepat.

2. REM (Relative to Root)

Ukuran rem relatif terhadap font-size elemen ROOT (html). Biasanya default browser adalah 16px.

  • 1rem biasanya = 16px.
  • 2rem = 32px.
  • **Kelebihan:** Sangat stabil dan memudahkan aksesibilitas. Jika user memperbesar font browser, seluruh website Anda akan ikut membesar secara proporsional.

Percentage (%)

Satuan persen selalu relatif terhadap elemen induknya (parent) untuk properti yang sama.

Contoh Penggunaan Lebar:

Jika induk lebarnya 100%, dan anak 50%, maka anak akan memakan setengah dari lebar induknya.

50% Lebar Induk

Kapan menggunakan %? Sangat berguna untuk membuat kolom layout yang bisa melebar atau mengecil sesuai ukuran layar.
