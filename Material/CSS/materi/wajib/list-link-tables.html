Menghapus Penanda (Sangat Penting untuk Navigasi!)

Seringkali kita ingin membuat menu navigasi menggunakan list tanpa bullet.

  • Beranda
  • Produk
  • Kontak
ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}

List Style Image

Anda bisa menggunakan gambar kustom sebagai bullet.

ul {
  list-style-image: url('sqpurple.gif');
}

CSS Links

Tautan dapat diberi gaya berbeda tergantung pada **statusnya** menggunakan *pseudo-classes*.

Coba arahkan kursor (hover) atau klik tautan ini: Contoh Tautan Interaktif

Empat Status Utama (LVHA):

  • :link - Tautan normal yang belum dikunjungi.
  • :visited - Tautan yang sudah pernah dikunjungi user.
  • :hover - Saat kursor berada di atas tautan.
  • :active - Saat tautan sedang diklik.
/* Urutan LVHA penting! */
a:link { color: blue; }
a:visited { color: purple; }
a:hover { color: red; }
a:active { color: yellow; }

CSS Tables

Tabel HTML tanpa CSS terlihat sangat berantakan. Properti CSS membuat data tabel mudah dibaca.

Properti Utama Tabel:

  • border-collapse: Menggabungkan border ganda menjadi satu garis tipis (Sangat penting!).
  • width & height: Menentukan ukuran tabel.
  • padding: Memberi ruang di dalam sel agar teks tidak menempel ke garis.
  • nth-child(even): Membuat efek "Zebra" (warna selang-seling).
