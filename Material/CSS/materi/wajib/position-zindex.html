CSS Position & Z-index

CSS Position

Properti position menentukan metode penempatan elemen di dokumen.

1. Static (Default)

Elemen mengikuti aliran normal halaman. Properti top, bottom, left, right **tidak berfungsi** pada static.

Static

2. Relative

Elemen dipindahkan relatif terhadap posisi aslinya tanpa mengganggu ruang yang ditinggalkannya.

Relative (left 50px)

3. Absolute

Elemen dipindahkan relatif terhadap **elemen induk (parent)** terdekat yang memiliki posisi (selain static). Jika tidak ada, ia merujuk ke body.

Absolute (bottom 10px)

4. Fixed

Elemen dipindahkan relatif terhadap **layar (viewport)**. Ia akan tetap di posisi yang sama meskipun halaman di-scroll (Lihat kotak ungu di pojok kanan bawah!).

Fixed Box

5. Sticky

Elemen bertingkah seperti relative sampai mencapai titik tertentu saat di-scroll, lalu menjadi fixed.

SAYA STICKY (Akan menempel saat scroll)

Z-index (Stacking Order)

Properti z-index menentukan urutan tumpukan elemen (elemen mana yang berada di depan/belakang elemen lain).

Penting: z-index hanya berfungsi pada elemen yang memiliki posisi (relative, absolute, fixed, atau sticky).

  • Nilai lebih besar = Berada lebih depan.
  • Nilai lebih kecil = Berada lebih belakang.

Contoh Tumpukan:

Z-index: 1
Z-index: 3
Z-index: 2

Pada contoh di atas, kotak Biru (z-index: 3) berada paling depan, diikuti Hijau (2), dan Merah (1) paling belakang.
