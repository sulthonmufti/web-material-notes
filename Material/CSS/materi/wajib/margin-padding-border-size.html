Margin, Padding, Border, & Size

Margin (Jarak Luar)

Margin digunakan untuk membuat ruang di **luar** elemen (di luar border).

Properti Individu

margin-top, margin-right, margin-bottom, margin-left.

Margin Shorthand

margin: 25px 50px 75px 100px; /* atas, kanan, bawah, kiri */
margin: 25px 50px;           /* atas&bawah 25px, kiri&kanan 50px */
margin: 25px;                /* semua sisi 25px */

Nilai "Auto"

Mengatur margin: auto; secara horizontal akan membuat elemen berada di tengah secara otomatis di dalam containernya (syarat: elemen harus memiliki width).

Kotak ini di tengah dengan margin: auto

Padding (Jarak Dalam)

Padding digunakan untuk membuat ruang di **dalam** elemen, antara konten dan border.

Padding Shorthand

padding: 20px 40px; /* 20px atas-bawah, 40px kiri-kanan */

Penting: Jika elemen memiliki warna latar belakang, warna tersebut akan terlihat memenuhi area padding.

Kotak dengan Padding Besar (40px)

Kotak dengan Padding Kecil (5px)

Border (Garis Tepi)

Border mengelilingi padding dan konten. Anda bisa mengatur lebar, gaya, dan warnanya.

Border Shorthand

border: 5px solid red; /* width, style, color */

Border Styles

Dotted Border
Dashed Border
Double Border

Border Radius (Sudut Tumpul)

Digunakan untuk membuat sudut elemen menjadi melengkung.

Kotak dengan border-radius: 15px

Height & Width (Ukuran)

Digunakan untuk menentukan tinggi dan lebar suatu elemen.

Nilai Ukuran

  • auto (default): Browser menghitung ukuran secara otomatis.
  • px: Ukuran tetap (pixel).
  • %: Ukuran relatif terhadap elemen induknya (parent).

Max-width (Penting untuk Responsif)

Gunakan max-width daripada width agar elemen tidak meluap keluar layar pada HP yang kecil.

Kotak ini lebarnya 100% tapi maksimal hanya 300px.
