CSS Display Property

Properti display adalah properti paling dasar untuk mengontrol layout. Setiap elemen memiliki nilai default tergantung jenis elemennya.

1. Display: Block

Elemen dengan display: block akan mengambil **seluruh lebar** yang tersedia dan selalu dimulai pada **baris baru**.

  • Dapat diatur width dan height-nya.
  • Margin dan Padding berlaku di semua sisi.
  • Contoh elemen default block: <div>, <h1>-<h6>, <p>, <section>.

2. Display: Inline

Elemen dengan display: inline hanya mengambil lebar **sesuai kebutuhan kontennya** saja dan **tidak** memulai baris baru.

  • **Tidak bisa** diatur width dan height-nya.
  • Margin/Padding atas dan bawah tidak akan mempengaruhi elemen lain di sekitarnya.
  • Contoh elemen default inline: <span>, <a>, <img>.

3. Display: Inline-Block

Ini adalah jalan tengah. Elemen akan berbaris ke samping (seperti inline), tetapi kita **bisa** mengatur width dan height-nya (seperti block).

  • Cocok untuk membuat menu navigasi atau kartu (cards) yang berjejer ke samping.

4. Display: None

Digunakan untuk **menghilangkan** elemen sepenuhnya dari halaman. Elemen tersebut tidak akan mengambil ruang sama sekali, seolah-olah tidak ada di HTML.

Bedanya dengan Visibility: Hidden?
display: none; -> Elemen hilang dan ruangnya tertutup.
visibility: hidden; -> Elemen tidak terlihat tapi ruangnya tetap kosong (seperti hantu). 
.tombol-rahasia {
  display: none;
}

Nilai Modern: Flex & Grid

Selain nilai di atas, ada nilai modern yang sangat kuat untuk layout:

  • display: flex; (Untuk layout satu dimensi).
  • display: grid; (Untuk layout dua dimensi/tabel kompleks).

Materi Flex dan Grid akan dibahas di bagian Lanjutan/Mahir.
