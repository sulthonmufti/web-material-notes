CSS Box Model (MOdel Kotak)

Box Model adalah konsep paling mendasar di CSS. Setiap elemen HTML dianggap sebagai sebuah **kotak** (box). Memahami struktur kotak ini sangat penting untuk penataan layout.

Komponen Dasar Box Model

Sebuah kotak terdiri dari empat lapisan, dari dalam ke luar.

  • 4. Margin (Jarak antar elemen)
  • 3. Border (Garis pembatas)
  • 2. Padding (Jarak dari konten ke border)
  • 1. Content (Konten aktual: teks, gambar, dll.)

Lihat contoh kotak di bawah. Semua area di luar garis putus-putus adalah **margin**.

CONTENT (250px x 100px)

1. Content (Konten)

  • **Deskripsi:** Area tempat konten sesungguhnya (teks, gambar) berada.
  • **Properti:** Dikontrol oleh properti width (lebar) dan height (tinggi).
  • **Contoh:** Pada kode di atas, koten memiliki lebar 250px dan tinggi 100px.

2. Padding

  • **Deskripsi:** Jarak /ruang kosong yang berada di antara **Content** dan **Border**. Padding bersifat transparan dan mengambil warna latar belakang (*background color*) elemen.
  • **Properti:** padding-top, padding-right, padding-bottom, padding-left, aa=tay *shortland padding.
  • **Contoh:** padding: 25px; (memberi jarak 25px di semua sisi).

/* Padding Shortland*/
padding: 10px;                  /*Semua sisi 10px*/
padding: 10px 20px;             /*Top/bottom 10px, left/right 20px*/
padding: 10px 20px 30px;         /*Top 10px, left/right 20px, bottom 30px*/
padding: 10px 20px 30px 40px;   /*Top, tight, bottom, left (searah jarum jam)/

3. Border (Garis tepi)

  • **Deskripsi:** Garis yang mengelilingi Padding dan Content. Border memisahkan elemen dari lingkungannya.
  • **Properti:** border-width, border-style, border-color, atau *shorthand* border.
  • **Contoh:** border: 10px solid #f39c12; (Ketebalan 10px, tipe solid, warna oranye).

4. Margin

  • **Deskripsi:** Ruang kosong yang berada di luar **Border**. Margin menciptakan jarak antara satu elemen dengan elemen di sekitarnya. Margin selalu transparan.
  • **Properti:** margin-top, margin-right, margin-bottom, margin-left, atau *shorthand* margin.
  • **Contoh:** margin: 20px; (Memberi jarak 20px dari elemen di sekitarnya).

Konsep Penting: Maegin Collapsing

Ketika dua elemen vertikal bersebelahan, margin atas dan bawahnya akan bergabung (*collapse*), dan jarak yang diambil hanya yang terbesar. Hal ini terjadi pada margin vertikal (atas/bawah).

Box Sizing (Cara Menghitung Ukuran Total)

Inilah bagian yang paling sering membuat pemula bingung.

1. Mode Default: box-sizing: content-box (Model Klasik)


Total Lebar = width + padding left + padding-right + border-left + border-right

Contoh di atas memiliki total lebar 250px (width) + 25px (padding kiri) + 25px (padding kanan) + 10px (border kiri) + 10px (border kanan) = **320px**

2. Model Modern: box-sizing: border-box (Model Terbaik)

Dalam mode ini, properti width dan height sudah **termasuk ** **padding** dan **border**. Margin tetap di luar.


Total Lebar = width (yang sudah termasuk padding dan border)

Dengan border-box, jika kita set width 250px, elemen akan bernar-benar selebar 250px. ini membuat penataan *layout* jauh lebih mudah dan prediktif. Hampir semua *framwork* dan proyek modern menggunakan model ini.


*{
    box-sizing: border-box; /* Terapkan ini di awal CSS*/
}
