Content-box

width content 300px + padding 20px + border 10px = 360px

note: box-sizing: content-box itu kaya belum fix karena belum ditambah padding border dll, misal content mu diisi width 300px tapi kamu tambahin padding 20px dan border 10px, maka keseluruhan content mu bakal jadi 360px. Secara deafult kalau kamau tidak mengatur box-sizing maka akan otomatis box-sizing: content-box


Border-box

width tetap 300px walau ditambah padding 20px & border 10px


note: box-sizing: border-box itu ukurannya udah fix, misal width kamu tentukan 300px walaupun ditambah padding 20px dan border 10px maka width nya akan tetap 300px. karena width itu akan dibagi bagi dulu. jadi width all 300px, border 10px, padding 20px sisanya masuk ke width content 240px

Programmer profesional akan lebih mengutamakan menggunakan box-sizing: border-box karena ukurannya pasti dan tidak akan berubah, terkhusus kalau kasusnya penggunaan width %
