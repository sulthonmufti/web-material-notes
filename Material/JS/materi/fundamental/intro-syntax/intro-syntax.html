JavaScript: Introduction & Syntax

Pintu Gerbang Menuju Frontend Interaktif

1. Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang membuat website "hidup". Jika HTML adalah struktur dan CSS adalah gaya, JS adalah perilakunya.

Kasus Penggunaan:

  • Mengubah isi teks tanpa reload.
  • Memvalidasi input di form.
  • Membuat pop-up atau notifikasi interaktif.

2. Cara Meletakkan Kode (Where To)

Ada dua cara yang sering dipakai di dunia kerja:

A. Internal Script

Langsung di dalam tag <script> sebelum penutup </body>.

B. External Script (Standar Industri)

Memisahkan kode JS ke file .js agar file HTML tetap rapi.

<!-- Di file index.html -->
<script src="script.js"></script>
