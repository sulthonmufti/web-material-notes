JS: Arrays Basics

Mengelola Koleksi Data dalam Satu Daftar

1. Pengenalan Array +

Array adalah variabel khusus yang dapat menampung lebih dari satu nilai sekaligus. Data dalam Array disusun secara berurutan menggunakan nomor urut yang disebut Index.

üìñ Istilah Baru: Index & Element
  • Index: Nomor urut posisi data (Dimulai dari 0).
  • Element: Nilai atau data yang ada di dalam Array. 
const buah = ["Apel", "Jeruk", "Mangga"];
// Index: 0 = Apel, 1 = Jeruk, 2 = Mangga
2. Mengakses & Mengubah Array +

Anda bisa mengambil atau mengubah data spesifik dengan memanggil nomor index-nya menggunakan kurung siku [].

Daftar Buah: ["Apel", "Jeruk", "Mangga"]

üõ†Ô∏è Dapur Rekaman: Proses Manipulasi

let item = buah[1]; // Mengambil "Jeruk"
buah[0] = "Melon"; // Mengganti "Apel" menjadi "Melon"

Penting: Ingatlah bahwa index selalu mulai dari 0. Jika Anda punya 3 data, maka index maksimalnya adalah 2.

3. Metode Dasar (Push, Pop, Shift, Unshift) +

Metode adalah fungsi bawaan JavaScript untuk memanipulasi daftar data tanpa harus mengubahnya secara manual.

Metode Kegunaan Posisi
push() Menambah data baru Akhir Array
pop() Menghapus data terakhir Akhir Array
unshift() Menambah data baru Awal Array
shift() Menghapus data pertama Awal Array

Daftar Nama: ["Ali", "Budi"]

üõ†Ô∏è Dapur Rekaman: Analisis Method

Push & Pop: Sangat efisien karena JavaScript tidak perlu mengubah nomor index elemen lainnya. Ini sering digunakan untuk fitur Undo/Redo atau riwayat pencarian.

4. Properti Length +

Properti length digunakan untuk mengetahui jumlah total elemen di dalam sebuah Array.

üí° Tips: Akses Data Terakhir Karena index mulai dari 0, data terakhir selalu berada di posisi array.length - 1. 
const hobi = ["Coding", "Gaming", "Reading"];
console.log(hobi.length); // Hasil: 3
