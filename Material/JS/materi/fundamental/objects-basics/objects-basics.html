JS: Objects Basics

Mengelompokkan Data Terkait dalam Satu Wadah

1. Pengenalan Object +

Object adalah variabel yang dapat menampung banyak nilai dalam format key: value. Jika variabel biasa menyimpan "Budi", maka Object bisa menyimpan "Budi", umur "25", dan pekerjaan "Developer" dalam satu nama.

üìñ Istilah Baru: Property & Value
  • Property (Key): Nama label di dalam object (seperti nama laci).
  • Value: Isi dari label tersebut.
const user = {
    nama: "Budi",
    umur: 25,
    pekerjaan: "Frontend Dev"
};
2. Cara Mengakses Data Object +

Ada dua cara utama untuk mengambil isi dari sebuah laci (property) di dalam object.

üìñ Istilah: Dot Notation & Bracket Notation
  • Dot Notation: user.nama (Paling sering digunakan).
  • Bracket Notation: user["nama"] (Digunakan jika nama properti mengandung spasi atau berupa variabel).

üõ†Ô∏è Dapur Rekaman: Kenapa ada dua cara?

Gunakan Dot Notation untuk kode yang bersih. Gunakan Bracket Notation jika Anda ingin mengakses properti secara dinamis (misal: nama laci disimpan dalam variabel lain).

3. Object Methods +

Object tidak hanya bisa menyimpan data, tapi juga bisa memiliki "kemampuan" atau fungsi. Fungsi yang ada di dalam object disebut Method. 

const mobil = {
    merk: "Toyota",
    starter: function() {
        return "Mesin menyala... brumm!";
    }
};

Klik untuk menjalankan method starter():

4. Mengenal Keyword 'this' +

Di dalam sebuah method, keyword this merujuk kepada Object pemiliknya. Ini digunakan agar object bisa mengakses datanya sendiri.

üõ†Ô∏è Dapur Rekaman: Analisis 'this'

const orang = {
    nama: "Andi",
    sapa: function() {
        return "Halo, nama saya " + this.nama;
    }
};

Proses: this.nama sama saja dengan memanggil orang.nama. JavaScript menggunakan this agar fungsinya tetap bekerja meskipun nama variabel object-nya diganti.
