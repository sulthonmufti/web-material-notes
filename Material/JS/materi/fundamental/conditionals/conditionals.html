2. Switch Statement +

Switch digunakan ketika Anda memiliki banyak kondisi yang membandingkan satu nilai yang sama dengan banyak pilihan (case). Lebih rapi daripada menulis else if yang terlalu banyak.

ğŸ“– Istilah: Break & Default
  • Break: Menghentikan pengecekan case lainnya setelah satu case terpenuhi.
  • Default: Kode cadangan jika tidak ada satupun case yang cocok (seperti else).

Pilih Warna Tema:

ğŸ› ï¸ Dapur Rekaman: Kenapa pakai Switch?

switch(warna) {
    case "Merah": 
        background = "red"; 
        break;
    default: 
        background = "transparent";
}
