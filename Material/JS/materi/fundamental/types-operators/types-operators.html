JS: Data Types & Operators

Memahami Bahan Baku dan Mesin JavaScript

1. Tipe Data Primitif +

JavaScript memiliki beberapa tipe data dasar. Yang paling sering Anda temui adalah:

  • String: Teks (dibungkus kutip).
  • Number: Angka (desimal atau bulat).
  • Boolean: Hanya dua nilai: true atau false.
  • Undefined: Variabel yang sudah dibuat tapi belum diisi.
  • Null: Nilai kosong yang disengaja.
üìñ Istilah: Dynamic Typing JavaScript adalah bahasa yang "dinamis". Artinya, satu variabel bisa berubah dari angka menjadi teks tanpa error. 
let nama = "Ali";      // String
let umur = 25;         // Number
let isLulus = true;    // Boolean

3. Perbandingan & Logika +

Digunakan untuk mengambil keputusan (If-Else).

  • == (Sama nilainya) vs === (Sama nilai & tipe datanya).
  • && (DAN), || (ATAU), ! (TIDAK).
üìñ Istilah: Strict Equality (===) Developer Frontend profesional selalu menyarankan === untuk menghindari bug "aneh" di JavaScript.

Apakah 5 == "5"?

Apakah 5 === "5"?

üõ†Ô∏è Dapur Rekaman: Kenapa Hasilnya Berbeda?

Loose Equality (==): Mencoba memaksa tipe data agar sama sebelum dibandingan. "Pokoknya angkanya 5, ya dianggap sama."

Strict Equality (===): Tidak ada toleransi. "Yang satu angka, yang satu teks? Beda!". Ini lebih aman untuk aplikasi besar.
