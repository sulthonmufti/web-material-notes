HTML Headings

Headings digunakan untuk mendefinisikan struktur konten Anda. Browser menggunakannya untuk membuat indeks halaman otomatis.

Istilah Penting:
  • Hierarki: Urutan kepentingan. <h1> paling penting, <h6> paling tidak penting.
  • SEO (Search Engine Optimization): Cara mesin pencari (Google) membaca isi web Anda.

Contoh Visual Hierarki:

Judul Utama (H1)

Sub-bab (H2)

Detail Poin (H3)

Kasus Nyata: Mengapa jangan asal pilih Heading?

Jangan gunakan Heading hanya untuk memperbesar teks. Gunakan Heading untuk struktur. Jika Anda ingin teks besar tapi bukan judul, gunakan CSS. Google akan menghukum website yang memiliki lebih dari satu <h1> karena dianggap strukturnya berantakan.

Paragraphs & Line Breaks

Bagaimana cara memisahkan teks agar tidak menjadi satu blok besar yang membosankan?

Tag yang Sering Dipakai:
  • <p>: Paragraph. Memberikan jarak (margin) otomatis.
  • <br>: Line Break. Pindah baris tanpa jarak tambahan (Self-closing tag).
  • <hr>: Horizontal Rule. Membuat garis pembatas antar konten.

Contoh Kode:

<p>Ini paragraf pertama.</p>
<hr>
<p>Ini baris satu.<br>Ini baris dua (tanpa jarak).</p>

Kasus Nyata: Alamat atau Puisi

Saat menulis alamat atau puisi, kita tidak ingin ada jarak besar antar baris. Di sinilah tag <br> sangat berguna dibanding membuat banyak tag <p>.

HTML Styles (Inline CSS)

Meskipun nantinya kita akan menggunakan file CSS terpisah, Anda bisa memberikan gaya langsung pada elemen menggunakan atribut style.

Sintaks Style:

<tag style="properti:nilai;">

Properti yang Sering Digunakan:

  • color: Mengubah warna teks.
  • background-color: Mengubah warna latar.
  • font-size: Mengubah ukuran teks.
  • text-align: Mengatur posisi teks (left, center, right).

Demo Visual:

Teks dengan latar biru muda.

Teks besar di tengah.

Kapan Menggunakan Inline Style?

Gunakan ini hanya untuk perubahan cepat atau elemen unik yang hanya muncul sekali. Jika Anda ingin semua paragraf berwarna biru, jangan gunakan ini satu per satu; gunakan file CSS eksternal agar lebih efisien.
