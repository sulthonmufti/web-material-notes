Introduction to HTML

HTML adalah bahasa standar yang digunakan untuk membuat kerangka halaman web. Tanpa HTML, browser tidak akan tahu cara menampilkan teks, gambar, atau link.

Istilah Penting:
  • Hypertext: Teks yang mengandung link ke teks lain.
  • Markup: Memberikan instruksi (tanda) pada teks biasa agar memiliki format tertentu.
  • Tag: Instruksi kode di dalam kurung siku, misal <h1>.

Analogi Sederhana:

Bayangkan membangun rumah. HTML adalah susunan batu bata dan semen (rangka), CSS adalah cat dan dekorasinya, sedangkan JS adalah sistem kelistrikannya.

Kapan Ini Digunakan?

Digunakan setiap kali Anda ingin membuat struktur website. Apapun framework-nya (React, Vue, etc), output akhirnya selalu berupa HTML agar bisa dibaca browser.

Struktur Dokumen HTML

Setiap file HTML yang valid harus memiliki "tulang punggung" standar agar browser bisa merender halaman dengan benar.

Kode Template Wajib:

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
    <p>Ini konten website Anda.</p>
</body>
</html>
Penjelasan Runtut:
  1. <!DOCTYPE html>: Memberitahu browser bahwa ini adalah HTML5 terbaru.
  2. <html>: Elemen akar yang membungkus seluruh konten.
  3. <head>: Tempat menyimpan data teknis (judul tab, CSS, meta).
  4. <body>: Tempat semua konten visual yang dilihat pengunjung.

HTML Editors

Meskipun Anda bisa menulis kode di Notepad, pengembang profesional menggunakan IDE (Integrated Development Environment) untuk bekerja lebih cepat.

Rekomendasi Editor:

  • Visual Studio Code (VS Code): Standar industri saat ini. Gratis dan sangat cerdas.
  • Sublime Text: Ringan, simpel, dan sangat cepat untuk komputer spek rendah.
  • Notepad++: Pilihan klasik yang jauh lebih baik daripada Notepad bawaan Windows.

Cara Praktik Langsung:

  1. Buka Editor Anda.
  2. Buat file baru, simpan dengan nama index.html.
  3. Ketik ! lalu tekan Tab (di VS Code) untuk memunculkan kode otomatis.
  4. Buka file tersebut di browser (Chrome/Edge).
