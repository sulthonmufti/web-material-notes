Anatomi Elemen HTML

Sebelum menulis banyak kode, Anda harus paham bagaimana satu baris kode HTML dibentuk.

Istilah Penting:
  • Start Tag: Tanda dimulainya elemen (contoh: <p>).
  • Content: Isi teks atau media di dalam tag.
  • End Tag: Tanda berakhirnya elemen, ditandai dengan garis miring (contoh: </p>).
  • Nested Elements: Elemen di dalam elemen (elemen bersarang).

Contoh Kode:

<p>Ini adalah konten paragraf.</p>

Aturan Emas: Jangan Lupa Tag Penutup!

Meskipun beberapa tag bisa berjalan tanpa penutup di browser modern, melewatkan tag penutup dapat merusak tata letak (layout) seluruh website Anda secara tidak terduga.

Headings & Paragraphs

Ini adalah cara utama kita memberikan informasi tekstual kepada pengguna.

1. Headings (h1 - h6)

Digunakan untuk judul dan sub-judul. <h1> adalah yang terbesar/terpenting, dan <h6> adalah yang terkecil.

Judul Tingkat 1

Judul Tingkat 2

Judul Tingkat 3

2. Paragraphs (p)

Digunakan untuk membungkus blok teks biasa. Browser otomatis memberikan sedikit ruang kosong (margin) di atas dan bawah paragraf.

<h1>Manfaat Belajar Coding</h1>
<p>Coding membantu melatih logika dan pemecahan masalah.</p>

Kasus Nyata: SEO (Search Engine Optimization)

Google menggunakan tag <h1> untuk memahami topik utama halaman Anda. Jangan pernah gunakan lebih dari satu <h1> dalam satu halaman agar peringkat Google Anda bagus!

Links & Images

Website tidak akan menjadi "Web" tanpa kemampuan untuk berpindah halaman dan menampilkan visual.

Istilah Baru: Attribute

Informasi tambahan yang diletakkan di dalam tag pembuka. Contoh: href untuk link dan src untuk gambar.

1. Links (anchor tag)

Menggunakan tag <a> dengan atribut href.

<a href="https://google.com">Kunjungi Google</a>

2. Images

Menggunakan tag <img>. Uniknya, tag ini **tidak punya tag penutup** (Self-closing).

<img src="gambar-kucing.jpg" alt="Foto Kucing Lucu" width="300">

Penting: Atribut "Alt" pada Gambar

Atribut alt (Alternative Text) sangat penting untuk aksesibilitas. Jika gambar gagal dimuat atau jika pengunjung adalah tunanetra yang menggunakan pembaca layar (screen reader), teks inilah yang akan dibacakan.
