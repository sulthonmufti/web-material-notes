Menambahkan CSS ke HTML

CSS (*Cascading Style Sheets*) digunakan untuk mengatur tampilan elemen HTML. Ada tiga cara untuk menerapkannya:

Istilah Penting:
  • Inline: Gaya ditaruh langsung di tag HTML.
  • Internal: Gaya ditaruh di dalam tag <style> pada bagian <head>.
  • External: Gaya ditaruh di file terpisah berakhiran .css.

Kenapa harus belajar ketiganya?

Meskipun External CSS adalah standar industri, Anda akan sering menggunakan Inline CSS untuk pengujian cepat atau saat bekerja dengan Email HTML yang membutuhkan gaya langsung di tempat.

Inline & Internal CSS

1. Inline CSS

Digunakan untuk menerapkan gaya unik pada satu elemen saja menggunakan atribut style.

<h1 style="color:blue; text-align:center;">Ini Judul Biru</h1>

2. Internal CSS

Digunakan jika satu halaman HTML memiliki gaya yang unik dan berbeda dari halaman lainnya.

<head>
  <style>
    body { background-color: linen; }
    h1 { color: maroon; margin-left: 40px; }
  </style>
</head>

Kapan Menggunakan Internal CSS?

Contohnya adalah halaman "Landing Page" promosi yang hanya ada satu jenis saja di website Anda. Anda tidak perlu membuat file CSS baru jika gayanya tidak akan dipakai lagi di halaman lain.

External CSS

Ini adalah cara yang paling efisien. Dengan satu file CSS, Anda bisa mengubah tampilan 100 halaman web sekaligus.

Langkah Persiapan:
  1. Buat file baru bernama style.css.
  2. Hubungkan di HTML menggunakan tag <link> di dalam <head>.

Contoh Penghubung:

<head>
  <link rel="stylesheet" href="style.css">
</head>

Analogi Nyata: Seragam Perusahaan

Jika perusahaan mengubah warna seragam, mereka tidak mendatangi karyawan satu per satu untuk menjahit ulang (Inline). Mereka mengubah desain di pusat (External file), dan semua karyawan (halaman HTML) otomatis mengikuti perubahan tersebut hanya dengan mengganti "link" referensinya.

Ringkasan Perbedaan:

Metode Prioritas Efisiensi
Inline Paling Tinggi Sangat Rendah
Internal Sedang Sedang
