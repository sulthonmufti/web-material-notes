Apa itu HTML Attributes?

Jika elemen HTML adalah "benda", maka atribut adalah "sifat" atau "instruksi tambahan" yang ditempelkan pada benda tersebut.

Aturan Penulisan:
  • Atribut selalu ditulis di dalam Start Tag (tag pembuka).
  • Biasanya berpasangan antara nama dan nilai (name="value").
  • Nilai atribut harus selalu berada di dalam tanda petik (kutip dua "" sangat direkomendasikan).

Contoh Kode Dasar:

<tag nama-atribut="nilainya">Konten</tag>

Kasus Nyata: Mengarahkan Link

Tanpa atribut href, sebuah tag <a> (link) tidak akan tahu ke mana ia harus pergi saat diklik. Atributlah yang memberikan tujuan tersebut.

Atribut yang Sering Dipakai

Hampir setiap tag HTML memiliki atribut spesifiknya sendiri. Berikut adalah yang paling sering muncul di W3Schools:

Atribut Digunakan Pada Fungsi
href <a> Menentukan URL tujuan link.
src <img>, <video> Menentukan sumber (path) file.
alt <img> Teks pengganti jika gambar rusak.
style Semua Elemen Menambahkan desain inline (warna, font).

Tips Visual: Atribut "Title"

Atribut title memberikan informasi tambahan berupa "tooltip" kecil saat Anda mengarahkan kursor (hover) ke arah elemen.

Dekatkan kursor Anda ke sini untuk melihat fungsi atribut title.

Global Attributes: ID & Class

Ada atribut yang bisa dipasang di semua elemen HTML. Yang paling penting untuk dipelajari sebelum masuk ke CSS dan JS adalah id dan class.

ID vs CLASS:
  • ID: Harus unik (hanya satu di seluruh halaman). Ibarat Nomor Induk Kependudukan (NIK).
  • Class: Bisa dipakai berkali-kali pada banyak elemen. Ibarat Seragam Sekolah.

Contoh Kode:

<!-- Menggunakan Class (Bisa banyak elemen) -->
<p class="teks-biru">Paragraf 1</p>
<p class="teks-biru">Paragraf 2</p>

<!-- Menggunakan ID (Hanya satu di halaman) -->
<h1 id="judul-utama">Selamat Datang</h1>

Kapan Menggunakan ID atau Class?

Gunakan Class jika Anda ingin memberikan gaya (styling) yang sama pada banyak elemen (misal: semua tombol berwarna merah). Gunakan ID jika Anda ingin menargetkan satu elemen spesifik untuk fungsi JavaScript tertentu.
