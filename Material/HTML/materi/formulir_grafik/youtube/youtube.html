YouTube Iframe

YouTube menggunakan tag <iframe> untuk menyisipkan video ke dalam website lain. Ini disebut teknik Embedding.

Cara Mendapatkan Kode:
  1. Buka video di YouTube.
  2. Klik tombol Share.
  3. Pilih Embed.
  4. Salin kode iframe yang muncul.

Struktur Kode Utama:

<iframe width="560" height="315" 
  src="https://www.youtube.com/embed/VIDEO_ID" 
  title="YouTube video player" 
  allowfullscreen>
</iframe>
Istilah Baru: VIDEO_ID

Kode unik (biasanya 11 karakter) yang muncul setelah v= di URL YouTube asli. Contoh: Jika URL-nya youtube.com/watch?v=dQw4w9WgXcQ, maka ID-nya adalah dQw4w9WgXcQ.

Parameter YouTube

Anda bisa mengontrol perilaku video dengan menambahkan parameter di ujung URL di dalam atribut src menggunakan tanda tanya (?).

Parameter Sering Dipakai:
  • autoplay=1: Video langsung jalan (harus dikombinasikan dengan mute=1).
  • mute=1: Mematikan suara saat pertama kali diputar.
  • loop=1: Mengulang video terus menerus (membutuhkan parameter playlist=ID_VIDEO).
  • controls=0: Menghilangkan tombol navigasi YouTube.

Contoh URL dengan Parameter:

src="https://www.youtube.com/embed/ID?autoplay=1&mute=1"

Kasus Nyata: Landing Page

Pada halaman produk, video sering diset autoplay=1&mute=1&loop=1 untuk menunjukkan cara kerja produk secara visual tanpa mengganggu pengguna dengan suara mendadak.

Video yang Fleksibel (Responsive)

Masalah utama iframe adalah ukurannya yang tetap (piksel). Di layar HP, video akan terpotong atau terlalu kecil.

Solusi CSS (Aspect Ratio):

Kita perlu membungkus iframe dengan <div> dan menggunakan teknik perbandingan 16:9 agar video mengikuti lebar layar.

Praktik Terbaik (HTML & CSS):

<!-- HTML -->
<div class="video-container">
  <iframe src="https://www.youtube.com/embed/ID"></iframe>
</div>

<!-- CSS Dasar -->
.video-container {
  position: relative;
  padding-bottom: 56.25%; /* Perbandingan 16:9 */
  height: 0;
}
.video-container iframe {
  position: absolute;
  top: 0; left: 0; width: 100%; height: 100%;
}
