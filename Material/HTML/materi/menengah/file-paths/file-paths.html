Absolute File Paths

Absolute file path adalah alamat lengkap dari sebuah file di internet. Ini seperti menuliskan alamat rumah lengkap dengan negara dan kode posnya.

Ciri Khas:
  • Selalu diawali dengan protokol (http:// atau https://).
  • Mengarah ke file yang berada di website lain atau server luar.
  • Link ini akan tetap bekerja selama website sumber tidak menghapus filenya.

Contoh Kode:

<img src="https://www.w3schools.com/images/picture.jpg" alt="Gunung">

Kapan Digunakan?

Gunakan absolute path jika Anda ingin mengambil sumber daya (gambar, library CSS, atau video) yang tidak Anda miliki sendiri, atau jika ingin membuat link ke website orang lain.

Relative File Paths

Relative file path menjelaskan letak sebuah file relatif terhadap posisi file HTML yang sedang dibuka saat ini. Ini adalah cara yang paling direkomendasikan untuk mengelola file website Anda sendiri.

Aturan Dasar:
  • Satu Folder: Jika file ada di folder yang sama, cukup tulis namanya.
    src="gambar.jpg"
  • Di dalam Folder: Jika file ada di dalam sub-folder.
    src="images/gambar.jpg"
  • Akar Website: Diawali dengan slash (/) untuk merujuk ke folder utama.
    src="/images/gambar.jpg"

Struktur Folder Contoh:

index.html
|__ images/
   |__ logo.png

Kodenya di index.html:

<img src="images/logo.png">

Mengapa Relative Path Lebih Baik?

Jika Anda memindahkan seluruh folder website Anda dari komputer ke server hosting, semua link akan tetap bekerja selama struktur folder di dalamnya tidak berubah. Jika pakai absolute path (C:/Users/Budi/...), link tersebut akan rusak di internet.

Navigasi Naik Folder (../)

Bagaimana jika file yang kita butuhkan berada "di luar" atau "di atas" folder tempat file HTML kita berada?

Simbol Ajaib: ../

Gunakan dua titik dan slash (../) untuk naik satu tingkat ke folder di atasnya.

Simulasi Struktur:

logo.png
|__ pages/
   |__ contact.html

Jika kita sedang mengedit contact.html dan ingin memanggil logo.png yang ada di luar folder 'pages':

<!-- Naik satu tingkat baru cari logo.png -->
<img src="../logo.png">

Kesalahan Umum

Lupa jumlah ../. Jika Anda perlu naik dua tingkat folder, Anda harus menulisnya dua kali: ../../images/foto.jpg. Pastikan Anda menghitung dengan teliti hirarki folder Anda!
