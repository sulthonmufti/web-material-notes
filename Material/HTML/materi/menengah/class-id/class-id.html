Atribut Class

Atribut class digunakan untuk menentukan satu atau lebih nama kelas untuk elemen HTML. Nama kelas ini bisa digunakan oleh banyak elemen sekaligus.

Ciri-ciri Class:
  • Dapat digunakan pada **banyak elemen** di satu halaman.
  • Satu elemen boleh memiliki **lebih dari satu class** (dipisahkan dengan spasi).
  • Dalam CSS, class dipanggil menggunakan tanda titik (.nama-class).

Contoh Kode:

<!-- Dua elemen dengan class yang sama -->
<p class="teks-biru">Paragraf pertama.</p>
<p class="teks-biru">Paragraf kedua.</p>

<!-- Satu elemen dengan dua class -->
<div class="kotak bayangan">Isi konten.</div>

Kasus Nyata: Desain Tombol

Jika Anda memiliki 10 tombol "Beli Sekarang" di sebuah website e-commerce, Anda cukup membuat satu class .btn-beli di CSS, lalu menempelkan class="btn-beli" ke semua tombol tersebut agar tampilannya seragam.

Atribut ID

Atribut id digunakan untuk menentukan identitas unik bagi sebuah elemen HTML. Nilai ID tidak boleh ada yang sama dalam satu halaman web.

Ciri-ciri ID:
  • Harus **unik** (hanya satu elemen yang boleh pakai nama ID tersebut).
  • Satu elemen hanya boleh memiliki **satu ID**.
  • Dalam CSS, ID dipanggil menggunakan tanda pagar (#nama-id).

Contoh Kode:

<h1 id="judul-utama">Selamat Datang di Web Saya</h1>

Kasus Nyata: Navigasi Anchor

ID sangat berguna untuk membuat link yang menuju ke bagian tertentu di halaman yang sama. Contoh: <a href="#kontak">Ke Kontak</a> akan otomatis menggulir halaman ke elemen yang memiliki id="kontak".

Class vs ID: Mana yang Digunakan?

Memilih antara Class dan ID sering membingungkan pemula. Gunakan tabel berikut sebagai panduan praktis:

Fitur Class ( . ) ID ( # )
Jumlah per halaman Bisa banyak Hanya satu (Harus unik)
Jumlah per elemen Bisa lebih dari satu Hanya satu
Kegunaan Utama Styling (CSS) Fungsi (JS) & Anchor
Aturan Penamaan:
  • Jangan diawali dengan angka.
  • Jangan gunakan spasi (gunakan tanda hubung - atau _).
  • Bersifat *Case-Sensitive* (Menu dan menu adalah hal yang berbeda).

Tips Pro: Prioritas

Jika sebuah elemen memiliki Class dan ID yang mengatur warna berbeda, maka **ID akan selalu menang** karena dianggap lebih spesifik oleh browser.
