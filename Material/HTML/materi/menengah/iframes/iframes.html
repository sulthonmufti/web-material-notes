Mengenal HTML Iframe

Iframe singkatan dari Inline Frame. Elemen ini digunakan untuk menyisipkan (embed) dokumen lain di dalam dokumen HTML yang sedang aktif.

Istilah Baru: Embed

Proses memasukkan konten dari situs luar (seperti video YouTube atau peta Google Maps) ke dalam website Anda sendiri tanpa harus mengunggah file tersebut ke server Anda.

Sintaks Dasar:

<iframe src="url_tujuan" title="deskripsi"></iframe>

Catatan Keamanan:

Beberapa situs besar seperti Google atau Facebook melarang situs mereka ditampilkan di dalam Iframe situs lain untuk mencegah serangan keamanan (Clickjacking). Jadi, jangan heran jika beberapa link tidak muncul saat dicoba di dalam iframe.

Atribut Penting Iframe

Agar tampilan iframe sesuai dengan desain web kita, ada beberapa atribut yang sering digunakan:

  • src: Alamat (URL) halaman yang ingin ditampilkan.
  • width & height: Menentukan ukuran jendela iframe (bisa dalam pixel atau persen).
  • title: Penting untuk aksesibilitas (menjelaskan isi iframe bagi pembaca layar).
  • loading="lazy": Menginstruksikan browser untuk memuat iframe hanya saat pengguna men-scroll ke bagian tersebut (menghemat kuota & mempercepat web).

Contoh Kode:

<iframe src="halaman-internal.html" 
        width="100%" 
        height="300" 
        title="Contoh Iframe">
</iframe>

Tips Visual: Menghapus Border

Secara default, browser memberikan border pada iframe. Gunakan CSS style="border:none;" di dalam tag iframe Anda untuk tampilan yang lebih menyatu dengan background.

Kasus Dunia Nyata

Iframe jarang digunakan untuk menampilkan seluruh halaman web lain, melainkan untuk fitur-fitur spesifik berikut:

Penggunaan Terpopuler:
  • YouTube: Menampilkan pemutar video langsung di web Anda.
  • Google Maps: Menampilkan lokasi toko atau kantor.
  • Iklan: Sebagian besar banner iklan di internet dimuat melalui iframe.

Demo Visual:

Keuntungan Menggunakan Iframe:

Konten di dalam iframe berjalan secara independen. Jika konten di dalam iframe lambat dimuat, hal itu tidak akan menghambat pemuatan konten utama di halaman web Anda.
