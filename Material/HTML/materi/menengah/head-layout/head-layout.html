The Head Element

Tag <head> adalah wadah untuk **Metadata**. Metadata adalah data tentang halaman HTML tersebut yang tidak ditampilkan langsung di layar browser, tetapi sangat penting bagi mesin pencari (SEO) dan browser.

Elemen di dalam Head:
  • <title>: Judul yang muncul di tab browser.
  • <meta>: Berisi deskripsi, kata kunci, penulis, dan pengaturan karakter.
  • <link>: Menghubungkan ke file luar (seperti CSS atau Favicon).
  • <script>: Menghubungkan ke file JavaScript.

Contoh Kode Head yang Ideal:

<head>
  <meta charset="UTF-8">
  <meta name="description" content="Tutorial HTML Lengkap">
  <title>Belajar HTML Layout - Materi 8</title>
  <link rel="stylesheet" href="style.css">
</head>

Kasus Nyata: Mengapa Title Penting?

Google menggunakan teks di dalam <title> sebagai link biru yang diklik orang di hasil pencarian. Jika Anda mengosongkannya, orang tidak akan tahu apa isi web Anda.

HTML5 Semantic Layout

Dulu, orang membuat tata letak hanya menggunakan <div>. Sekarang, HTML5 memperkenalkan elemen **Semantik** yang memiliki makna jelas bagi browser dan perangkat bantu (seperti pembaca layar untuk tunanetra).

Elemen Layout Utama:
  • <header>: Bagian kepala (logo, judul).
  • <nav>: Kumpulan link navigasi.
  • <main>: Konten utama halaman (hanya boleh satu per halaman).
  • <aside>: Konten samping (iklan, link tambahan).
  • <footer>: Bagian kaki (copyright, info kontak).

Simulasi Visual Struktur Web:

HEADER
NAV (MENU)
MAIN CONTENT
ASIDE
FOOTER

Kenapa Tidak Pakai Div Saja?

Menggunakan tag semantik membuat kode Anda lebih rapi ("bersih") dan membantu Google memahami bagian mana yang merupakan menu dan bagian mana yang merupakan isi berita utama.

Viewport & Responsive Dasar

Pernahkah Anda membuka website di HP dan tulisannya sangat kecil sehingga harus di-zoom? Itu karena pengaturan Viewport yang salah.

Istilah Baru: Viewport

Area tampilan web yang terlihat oleh pengguna. Di HP, viewport jauh lebih kecil daripada di laptop.

Tag Meta Viewport Wajib:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Penjelasan Atribut:

  • width=device-width: Mengatur lebar halaman mengikuti lebar layar perangkat.
  • initial-scale=1.0: Mengatur level zoom awal saat halaman pertama kali dimuat.

Kasus Nyata: Mobile-First

Hampir 60% trafik internet saat ini berasal dari perangkat mobile. Tanpa baris kode meta viewport di atas, website Anda akan dianggap "tidak layak" oleh Google untuk muncul di hasil pencarian mobile.
