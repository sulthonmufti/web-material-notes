Advanced Links

Link bukan sekadar pindah halaman. Anda bisa mengatur bagaimana browser membukanya dan ke bagian mana link tersebut mengarah.

Atribut Baru:
  • target="_blank": Membuka link di tab baru.
  • target="_self": Membuka di tab yang sama (default).
  • id & href=#id: Membuat "Bookmark" (navigasi ke bagian tertentu dalam halaman yang sama).

Contoh Navigasi Tab Baru:

<a href="https://google.com" target="_blank">Buka Google di Tab Baru</a>

Kasus Nyata: External vs Internal Link

Praktik terbaik UX adalah menggunakan target="_blank" untuk link yang mengarah ke luar website Anda (External), agar pengunjung tidak benar-benar meninggalkan web Anda. Untuk navigasi antar halaman di web sendiri (Internal), biarkan menggunakan tab yang sama.

Images & File Paths

Masalah tersulit bagi pemula saat memasukkan gambar adalah menentukan "lokasi" atau Path file yang benar.

Istilah Penting:
  • Absolute Path: URL lengkap gambar dari internet (Contoh: https://site.com/foto.jpg).
  • Relative Path: Lokasi gambar relatif terhadap file HTML Anda (Contoh: images/foto.jpg).

Penulisan Path yang Benar:

<!-- Gambar di folder yang sama -->
<img src="foto.jpg" alt="Deskripsi">

<!-- Gambar di dalam sub-folder bernama 'img' -->
<img src="img/foto.jpg" alt="Deskripsi">

Tips: Selalu Gunakan "alt"

Jika gambar gagal dimuat karena koneksi internet lambat, teks di dalam atribut alt akan muncul. Ini menjaga agar pesan di dalam gambar tetap tersampaikan.

Menjadikan Gambar Sebagai Link

Anda bisa menggabungkan kedua elemen ini untuk membuat tombol bergambar, banner iklan, atau logo navigasi.

Konsep:

Cukup bungkus tag <img> di dalam tag <a>.

Contoh Kode:

<a href="https://google.com">
    <img src="logo-google.png" alt="Google" style="width:100px;">
</a>

Kasus Nyata: Tombol Media Sosial

Hampir semua website menggunakan teknik ini untuk ikon Instagram, Facebook, dan WhatsApp di bagian bawah (footer) mereka.
