HTML Geolocation

HTML Geolocation API digunakan untuk mendapatkan posisi geografis pengguna. Karena ini masalah privasi, posisi **tidak akan tersedia** kecuali pengguna menyetujuinya.

Prasyarat Penting:
  • HTTPS: Geolocation hanya bekerja pada situs yang aman (HTTPS). Di localhost (saat belajar), fitur ini tetap aktif.
  • User Permission: Browser akan otomatis memunculkan pop-up "Allow" atau "Block".

Kasus Penggunaan:

  • Menemukan toko terdekat dari lokasi Anda.
  • Memberikan prakiraan cuaca lokal secara otomatis.
  • Menampilkan rute di peta (Google Maps).

Mendapatkan Lokasi

Metode utama yang digunakan adalah getCurrentPosition(). Metode ini mengembalikan koordinat Latitude (garis lintang) dan Longitude (garis bujur).

[Image of latitude and longitude coordinates on a globe]
Istilah Baru:
  • Latitude: Posisi utara-selatan dari garis khatulistiwa.
  • Longitude: Posisi timur-barat dari garis meridian.
  • Accuracy: Tingkat keakuratan lokasi dalam satuan meter.

Contoh Kode Dasar:

<script>
function getLocation() {
  if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(showPosition);
  } else {
    alert("Geolocation tidak didukung browser ini.");
  }
}

function showPosition(position) {
  console.log("Lat: " + position.coords.latitude);
  console.log("Lon: " + position.coords.longitude);
}
</script>

Menangani Masalah (Error Handling)

Tidak selamanya pengambilan lokasi berhasil. Kita harus menyiapkan pesan jika sesuatu salah terjadi.

Penyebab Umum Error:
  • Permission Denied: Pengguna menekan tombol "Block".
  • Position Unavailable: Sinyal GPS lemah atau tidak ada koneksi internet.
  • Timeout: Waktu pengambilan lokasi terlalu lama.

Kode Penanganan Error:

function showError(error) {
  switch(error.code) {
    case error.PERMISSION_DENIED:
      return "Pengguna menolak izin lokasi.";
    case error.POSITION_UNAVAILABLE:
      return "Informasi lokasi tidak tersedia.";
    case error.TIMEOUT:
      return "Waktu permintaan habis.";
  }
}

Tips Pro: watchPosition()

Jika Anda membuat aplikasi navigasi (seperti ojek online), gunakan watchPosition(). Bedanya, ia akan terus memperbarui lokasi setiap kali posisi pengguna berubah secara otomatis tanpa harus klik tombol lagi.
