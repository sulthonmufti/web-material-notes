Mengenal Web Storage

Web Storage memungkinkan aplikasi web menyimpan data secara lokal di dalam browser pengguna dalam bentuk pasangan Key (kunci) dan Value (nilai).

Dua Jenis Web Storage:
  • localStorage: Menyimpan data tanpa batas waktu. Data tidak akan terhapus meskipun browser ditutup atau komputer dimatikan.
  • sessionStorage: Menyimpan data sementara. Data akan langsung terhapus begitu tab browser atau jendela browser ditutup.

Kasus Penggunaan:

  • Menyimpan preferensi Dark Mode (Tema Gelap).
  • Menyimpan item di dalam Keranjang Belanja sebelum checkout.
  • Menyimpan skor tertinggi (High Score) pada game berbasis web.

Metode Web Storage

Kedua jenis storage memiliki cara kerja yang sama persis dalam hal penulisan kode:

Fungsi yang Sering Digunakan:
  • setItem(key, value): Menambahkan data baru.
  • getItem(key): Mengambil data berdasarkan kuncinya.
  • removeItem(key): Menghapus satu data spesifik.
  • clear(): Menghapus semua data yang ada di storage.

Contoh Kode JavaScript:

// Menyimpan data
localStorage.setItem("username", "Budi");

// Mengambil data
let nama = localStorage.getItem("username");

// Menghapus data
localStorage.removeItem("username");

Keamanan & Privasi:

Meskipun praktis, jangan pernah menyimpan data sensitif seperti Password atau Nomor Kartu Kredit di Web Storage. Data ini bisa diakses dengan mudah oleh siapa pun yang bisa membuka konsol browser di komputer tersebut.
