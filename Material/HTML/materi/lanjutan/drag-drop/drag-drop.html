Drag and Drop Dasar

Fitur Drag and Drop (Tarik dan Lepas) memungkinkan pengguna berinteraksi dengan elemen secara lebih intuitif.

Langkah Utama:
  • draggable="true": Atribut wajib agar elemen bisa ditarik.
  • DataTransfer: "Kantong" tempat menyimpan data elemen yang sedang ditarik.

Sintaks Dasar:

<!-- Membuat gambar bisa ditarik -->
<img src="img.jpg" draggable="true" ondragstart="drag(event)">

Kasus Nyata: Trello & KanBan

Aplikasi seperti Trello menggunakan fitur ini agar pengguna bisa memindahkan kartu tugas dari kolom "To-Do" ke "Done" dengan satu gerakan mouse.

Memahami Drag Events

Ada beberapa kejadian (event) yang terjadi selama proses tarik-lepas berlangsung:

Istilah & Fungsi:
  • ondragstart: Terjadi saat user mulai menarik elemen.
  • ondragover: Terjadi saat elemen berada di atas area drop. (Penting: Harus melakukan event.preventDefault() agar drop bisa terjadi).
  • ondrop: Terjadi saat elemen dilepaskan di area tujuan.

Logika JavaScript:

function allowDrop(ev) {
  ev.preventDefault(); // Mengizinkan drop
}

function drag(ev) {
  ev.dataTransfer.setData("text", ev.target.id);
}

function drop(ev) {
  ev.preventDefault();
  var data = ev.dataTransfer.getData("text");
  ev.target.appendChild(document.getElementById(data));
}

Simulasi Interaktif

Coba tarik kotak hijau di bawah ini dan pindahkan ke kotak kosong lainnya.

TARIK

Tips Penting:

Browser secara default melarang data dijatuhkan (drop) ke elemen lain. Itulah sebabnya fungsi event.preventDefault() pada event ondragover sangat krusial. Tanpanya, fitur drop tidak akan pernah bekerja.
