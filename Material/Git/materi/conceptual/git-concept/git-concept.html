2. Arsitektur: Tiga Area Utama Git +

Git bekerja dengan memindahkan data di antara tiga area: Working Directory, Staging Area, dan Repository.

ğŸ“– Istilah: Area Kerja
  • Working Directory: File yang sedang Anda edit (VS Code).
  • Staging Area: "Keranjang" untuk memilih perubahan mana yang akan disimpan.
  • Local Repo: "Brankas" tempat menyimpan permanen (Commit).
