Git: Installation & Setup

Mempersiapkan Senjata Sebelum Terjun ke Medan Perang

1. Apa yang Perlu Dipersiapkan? +

Sebelum menginstal Git, pastikan Anda memiliki dua "senjata" utama pengembang:

  • Text Editor: Sangat disarankan menggunakan Visual Studio Code (VS Code).
  • Terminal/CLI: Tempat kita mengetik perintah Git. (Command Prompt/PowerShell di Windows, atau Terminal di Mac/Linux).

3. Verifikasi Instalasi (Interactive Terminal) +

Setelah instalasi selesai, Anda harus memastikan Git sudah terdeteksi oleh sistem.

Simulasi: Ketik perintah git --version untuk cek versi Git di bawah ini:

user@computer:~$

Tips: Jika muncul tulisan "git is not recognized", artinya instalasi gagal atau PATH belum terdaftar. Anda harus mengulang instalasi atau restart komputer.

4. Konfigurasi Awal (Wajib!) +

Git perlu tahu siapa yang membuat perubahan. Anda harus mengatur nama dan email Anda sekali saja di awal.

# Atur Nama Pengguna
git config --global user.name "Nama Anda"

# Atur Email
git config --global user.email "email@anda.com"
